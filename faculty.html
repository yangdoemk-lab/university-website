คณะในมหาวิทยาลัยศิลปากร วิทยาเขตสนามจันทร์

🎓 คณะศึกษาศาสตร์

คำอธิบาย: ผลิตครูและบุคลากรทางการศึกษา

สาขา: ภาษาไทย / ภาษาอังกฤษ / ปฐมวัย / ประถมศึกษา / เทคโนโลยีการศึกษา

🔬 คณะวิทยาศาสตร์

คำอธิบาย: ศึกษาวิทยาศาสตร์พื้นฐานและประยุกต์

สาขา: คณิตศาสตร์ / ฟิสิกส์ / เคมี / ชีววิทยา / เทคโนโลยีชีวภาพ / วิทยาการคอมพิวเตอร์

⚙️ คณะวิศวกรรมศาสตร์และเทคโนโลยีอุตสาหกรรม

คำอธิบาย: ออกแบบ พัฒนาเครื่องจักร และเทคโนโลยีอุตสาหกรรม

สาขา: วิศวกรรมเครื่องกล / ไฟฟ้า / เคมี / อุตสาหการ / วัสดุ / เมคคาทรอนิกส์

🔎 คณะอักษรศาสตร์

คำอธิบาย: เป็นคณะที่เน้นการศึกษาด้านภาษา วัฒนธรรม มนุษยศาสตร์ และการสื่อสาร

สาขา: กลุ่มภาษา/ กลุ่มมนุษยศาสตร์ & สังคม / กลุ่มข้อมูล / ความรู้ / กลุ่มศิลปะการแสดง /กลุ่มพัฒนาสังคม

💻 คณะเทคโนโลยีสารสนเทศและการสื่อสาร (ICT)

คำอธิบาย: การสร้างสรรค์สื่อผสมผสานทั้งศิลปะและเทคโนโลยี

สาขา: เทคโนโลยีสื่อสร้างสรรค์ / นิเทศศาสตร์ / เทคโนโลยีดิจิทัลเพื่อธุรกิจ/การออกแบบ

🎼 คณะดุริยางคศาสตร์

คำอธิบาย:สายดนตรีแบบมืออาชีพไม่ได้เรียนแค่เล่นดนตรี แต่รวมถึง “ธุรกิจดนตรี + การผลิต + เทคโนโลยี”

สาขา: ดนตรีสากล / ดนตรีแจ๊ส / ดนตรีเชิงพาณิชย์ / ดนตรีประกอบสื่อ / เทคโนโลยีดนตรี & การผลิตเสียง / การประพันธ์เพลง

💊 คณะเภสัชศาสตร์

คำอธิบาย: แต่เรียนทั้ง วิทยาศาสตร์พื้นฐาน + การผลิตยา + การใช้ยา + การบริหารจัดการ

สาขา: เคมีเภสัชกรรม / ชีวเภสัชกรรม หรือ ชีววิทยา / การผลิตและควบคุมคุณภาพยา / เภสัชวิทยาและการรักษาโรค / การบริหารจัดการยาและโรงพยาบาล / การวิจัยยาใหม่และการพัฒนายา

🖌️ คณะจิตรกรรม ประติมากรรม และภาพพิมพ์(เรียนปี 1 ที่สนามจันทร์ แล้วย้ายไปเรียนปี 2-5 ที่วังท่าพระ (กรุงเทพฯ))

คำอธิบาย: เป็นการเรียน “ศิลปะเชิงลึก” ทั้งด้านเทคนิค + ความคิด

สาขา: จิตรกรรม (วาดภาพ) / ประติมากรรม (ปั้น) / ภาพพิมพ์ / ศิลปะไทย & สื่อผสม

🏫 คณะมัณฑนศิลป์ (เรียนปี 1 ที่สนามจันทร์ แล้วย้ายไปเรียนปี 2-4 ที่วังท่าพระ (กรุงเทพฯ))

คำอธิบาย: เป็นคณะสาย Design (ออกแบบ)

สาขา: นิเทศศิลป์ (Graphic Design) / ออกแบบภายใน (Interior Design) / ออกแบบผลิตภัณฑ์ / แฟชั่น / เซรามิก

🏺 คณะโบราณคดี (บางคนอาจลงพื้นฐานการเรียนปี 1 ที่สนามจันทร แล้วย้ายไปเรียนปี 2-4 ที่วังท่าพระ (กรุงเทพฯ))

คำอธิบาย: ไม่ใช่แค่ “ขุดของเก่า” อย่างที่หลายคนเข้าใจแต่คือการศึกษา มนุษย์ในอดีตแบบลึกมาก

สาขา: โบราณคดี / ประวัติศาสตร์ศิลปะ / มานุษยวิทยา

🏠 คณะสถาปัตยกรรมศาสตร์ (เรียนปี 1 ที่สนามจันทร์ แล้วย้ายไปเรียนปี 2-5 ที่วังท่าพระ (กรุงเทพฯ))

คำอธิบาย: ไม่ใช่แค่ “วาดบ้าน” แต่เรียน ออกแบบทุกด้านของสภาพแวดล้อมมนุษย์

สาขา: สถาปัตย์ / ออกแบบเมือง / ออกแบบภายใน
